# Backend Portfolio

This repository contains my backend projects built primarily with **PHP** and **vanilla JavaScript**.
The goal of these projects is to practice backend development, build robust APIs, work with databases, and showcase my server-side skills.

---

## Tech Stack

* PHP (procedural & OOP)
* Vanilla JavaScript
* MySQL / MariaDB
* RESTful APIs
* Authentication & Authorization
* MVC / Custom architecture
* HTML5 & CSS3 (when needed)
* Git & GitHub

---

## Projects

| Project        | Description                                            | Tech           |
| -------------- | ------------------------------------------------------ | -------------- |
| Webshop        | A small, educational webshop backend where I show how to put together a simple product list, shopping cart, and order logic from scratch, using a full vanilla stack (PHP + a little JS + plain HTML/CSS). | PHP, JS, MySQL | 
| Jegyertekesito | Mini ticket sales system where tickets for events can be purchased/reserved. | PHP, JS, MySQL | 
| Hotel          | This project presents the backend of a simple hotel reservation system. | PHP, JS, MySQL |

> Each project is located in its own folder and includes a dedicated README with setup instructions and technical details.

---

## Getting Started

Clone the repository:

```bash
https://github.com/doomhyena/Backend-portfolio.git
```
